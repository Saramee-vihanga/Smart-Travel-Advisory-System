# Prolog Project

A simple Prolog-based project developed using logic programming concepts and rule-based reasoning.

## 📌 Project Description

This project was created to demonstrate how Prolog can be used to solve logical problems using facts, rules, and queries.

The system can:
- Process logical rules
- Answer queries
- Perform reasoning based on given facts
- Demonstrate AI and expert system concepts

## 🚀 Features

- Logic-based problem solving
- Rule-based decision making
- User query handling
- Simple and efficient Prolog implementation

## 🛠 Technologies Used

- Prolog
- SWI-Prolog
